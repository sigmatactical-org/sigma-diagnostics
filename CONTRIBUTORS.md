# Contributors

## Sigma Tactical Group

Current maintenance of **Sigma Tactical CAN Viewer** (`can-viewer`).

- Repository: https://github.com/sigmatactical-org/can-viewer
- Stack: Slint UI, [dbc-rs](https://github.com/sigmatactical-org/dbc-rs), [mdf4-rs](https://github.com/sigmatactical-org/mdf4-rs)

## Upstream lineage

**can-viewer** was originally developed as a Tauri + TypeScript application. That work is gratefully acknowledged. The 0.3.0 release replaces the WebView frontend with a native Slint UI while preserving the Rust domain logic and `can-viewer` crate name.

Earlier standalone development of **dbc-rs** and related bounded collection work remains upstream lineage under MIT OR Apache-2.0.
