# Contributors

## Sigma Tactical Group

Current maintenance of **Sigma Diagnostics** (`sigma-diagnostics`).

- Repository: https://github.com/sigmatactical-org/sigma-diagnostics
- Stack: Slint UI, [dbc-rs](https://github.com/sigmatactical-org/dbc-rs), [mdf4-rs](https://github.com/sigmatactical-org/mdf4-rs)

## Upstream lineage

This project was originally published as **can-viewer** (Tauri + TypeScript). That work is gratefully acknowledged. The 0.3.0 release replaces the WebView frontend with a native Slint UI; the repository was later renamed **sigma-diagnostics** (shared domain crate of the same name).

Earlier standalone development of **dbc-rs** and related bounded collection work remains upstream lineage under MIT OR Apache-2.0.
